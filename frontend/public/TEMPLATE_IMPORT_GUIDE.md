# Panduan Import Excel - Diferensiasi Misi

## Format File Excel

Untuk import data diferensiasi misi, file Excel Anda harus memiliki format sebagai berikut:

### Kolom yang Diperlukan:
1. **unit_kerja** - Unit organisasi (misal: "PT (Perguruan Tinggi)", "UPPS (Unit Pengelola Program Studi)", "Keilmuan PS (Program Studi)")
2. **tipe_data** - Jenis data (misal: "Visi" atau "Misi")
3. **konten** - Isi teks dari visi atau misi

### Contoh Data:

| unit_kerja | tipe_data | konten |
|------------|-----------|--------|
| PT (Perguruan Tinggi) | Visi | Menjadi perguruan tinggi yang unggul dalam pengembangan teknologi dan inovasi |
| UPPS (Unit Pengelola Program Studi) | Misi | Menyelenggarakan pendidikan berkualitas tinggi dalam bidang teknologi informasi |
| Keilmuan PS (Program Studi) | Visi | Program studi yang menghasilkan lulusan kompeten di bidang TI |

## Cara Menggunakan:

1. **Download Template** - Klik tombol "Template" untuk mendapatkan file contoh
2. **Isi Data** - Tambahkan data Anda sesuai format di atas
3. **Simpan File** - Simpan dengan format .xlsx atau .xls
4. **Upload** - Klik tombol "Import Excel" dan pilih file Anda
5. **Verifikasi** - Sistem akan menampilkan pesan jika berhasil atau ada error

## Catatan Penting:

- Gunakan Header Kolom yang SAMA PERSIS: `unit_kerja`, `tipe_data`, `konten`
- Minimal ada 1 baris data (selain header)
- File harus berformat Excel (.xlsx atau .xls)
- Semua field harus terisi (tidak boleh kosong)
- Pastikan tidak ada karakter khusus yang tidak perlu

## Troubleshooting:

- **Error "Kolom yang hilang"** - Pastikan nama kolom sesuai dengan yang diminta
- **Error "File Excel kosong"** - Pastikan ada data dalam sheet
- **Import tidak berhasil** - Cek format data dan pastikan tidak ada typo pada header
